Changelog

Current version: {{APP_VERSION}}

v{{APP_VERSION}}

Orthographic notes & inscriptional spelling aid

  • Added Orthographic note section to each result card, displayed between Matched rules and Alignment.
  • Result cards for Ancient Greek now show orthographic correspondence notes — e.g. παιδίο is annotated as potentially corresponding to παιδίου (paidiou).
  • Added pre-403/2 BCE Attic spelling note kind: forms that may reflect pre-reform Attic inscriptional orthography are identified when data or orthography_hint supports it.
  • Added reading aid note kind to help students map inscriptional forms to dictionary headwords.
  • New orthography_hint parameter on POST /search — accepted values: "standard", "inscriptional", "pre_403_2_attic".
  • Search hit responses include a new orthographic_notes array (empty by default; does not break existing clients).
  • EN/JA translations added for all orthographic note labels and messages.
  • Orthographic note data is marked provisional and not yet expert-reviewed.

v0.1.x

Earlier changes

  • Search responses now include lexicon, distance matrix, and rule version metadata in data_versions.
  • Footer now reports the running application version with a link to this page.
  • Added this changelog page at /changelog.
  • Improved distance calculation fallbacks to support "lexicon-only" and "unigram-only" strategies when needed.
  • Added strict validation for distance matrix completeness and symmetry in the generation pipeline.
  • Improved error messages and type hints throughout the phonology package.
