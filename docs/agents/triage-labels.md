# Triage labels

Engineering skills use five canonical triage roles. Map them to these local
Markdown `Status:` values.

| Canonical role | Local status | Meaning |
| --- | --- | --- |
| `needs-triage` | `要評価` | Maintainer evaluation is required |
| `needs-info` | `情報待ち` | More information is required from the reporter |
| `ready-for-agent` | `エージェント対応可` | Fully specified and ready for an agent |
| `ready-for-human` | `人間対応要` | Human implementation or judgment is required |
| `wontfix` | `対応しない` | The issue will not be actioned |

When a skill refers to a canonical role, write the corresponding local status
in the issue file.
